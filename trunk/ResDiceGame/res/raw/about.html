Catan Dice Game for Android

This is an unofficial implementation of Catan Dice Game for Android. More information on the game can be found on boardgamegeek.com.

Graphics for this version are from a fan-made playsheet by James Chapman using art assets from the Mayfair versions of Catan (4th Ed.) and box art for the Catan Dice Game.

This program is open source. Visit us here to enter bugs, enhancements or to help program.
